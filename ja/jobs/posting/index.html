For Companies

求人掲載希望の
企業様へ

Nihon Gateway Germanyでは、ドイツ在住の日本語話者、日本語人材、日本企業に関心のある求職者へ向けた求人情報を掲載しています。 現在は掲載件数拡大のため、求人掲載を無料で受け付けています。

メールで掲載を相談する 掲載中の求人を見る
現在はPhase 1として、掲載件数拡大のため無料掲載を実施しています。無料掲載は予告なく終了・変更される場合があります。 掲載可否、掲載期間、表示順位、内容修正については運営側の判断により調整される場合があります。

現在の掲載料金

初期フェーズでは、求人情報を増やし、求職者にとって価値のあるページを作ることを優先しています。 そのため、一定期間は求人掲載を無料で受け付けます。

Phase 1 限定
€0 / 1求人

通常掲載、求人一覧への表示、基本情報の掲載を無料で受け付けます。

  • 無料掲載は試験運用期間中の暫定条件です。
  • 将来的に有料プラン、上位表示プラン、SNS告知プラン等を導入する可能性があります。
  • 無料期間中に掲載された求人についても、掲載継続・更新・表示順位は今後の運用方針により変更される場合があります。
  • 掲載内容が不適切、確認困難、またはサイト方針に合わない場合は掲載できない場合があります。

掲載できる求人

ドイツで生活する日本語話者、日本語を使って働きたい方、日系企業・日本関連ビジネスに関心のある方へ向けた求人を想定しています。

対象地域 ドイツ国内勤務、ドイツ在住者向け、またはドイツから応募可能な求人。
対象職種 事務、営業、物流、IT、飲食、販売、通訳、カスタマーサポートなど。
雇用形態 正社員、パート、アルバイト、ワーキングホリデー、学生向け求人など。

掲載までの流れ

掲載希望内容を送信

メールで会社名、求人タイトル、勤務地、雇用形態、給与目安、応募条件などをお送りください。

内容確認

運営側で求人内容、表記、掲載可否を確認します。必要に応じて修正確認をお願いする場合があります。

求人ページへ掲載

確認後、求人一覧ページに掲載します。掲載後も内容修正の相談が可能です。

掲載時に送っていただきたい情報

  • 会社名
  • 担当者名
  • 連絡先メールアドレス
  • 求人タイトル
  • 勤務地、都市名
  • 雇用形態
  • 給与目安
  • ビザサポートの有無
  • 求人概要
  • 職務詳細
  • 必要要件、歓迎条件
  • 会社URL、応募URL、または応募用メールアドレス
  • 会社ロゴ画像URLがあれば任意

掲載できない可能性がある内容

  • 違法性のある求人、または法令遵守が確認できない求人
  • 給与、勤務条件、勤務地、雇用形態が極端に不明確な求人
  • 過度な成果報酬のみの求人
  • 差別的、攻撃的、誤解を招く表現を含む求人
  • 実態確認が難しい事業者、または連絡先が不明確な求人
  • 求職者に不利益を与える可能性が高いと運営側が判断した求人

Phase 1 の方針

  • まずは求人掲載数を増やす
  • 日本語話者向け求人の検索性を高める
  • 企業・求職者双方の使いやすさを検証する

将来の有料化について

Phase 1終了後、通常掲載、上位表示、SNS告知、特集記事などの有料プランを導入する可能性があります。 その場合は掲載条件をページ上で案内します。
