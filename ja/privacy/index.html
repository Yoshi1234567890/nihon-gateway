Privacy Policy

プライバシーポリシー

本プライバシーポリシーは、Nihon Gateway Germany（以下「本サイト」）における個人情報、Cookie、アクセス解析、外部サービス、問い合わせ、会員機能、投稿機能等に関する取扱い方針を説明するものです。

1. 基本方針

本サイトは、利用者のプライバシーを尊重し、適用される法令および合理的な実務基準に従って情報を取り扱います。本サイトはドイツおよびEU圏の利用を想定しており、Cookieや解析ツールの利用に関しては、必要に応じて同意取得を前提とした運用を行います。

2. 取得する情報

本サイトでは、以下の情報を取得する場合があります。

  • お問い合わせ時に入力または送信される氏名、メールアドレス、件名、本文等
  • 会員登録時の登録情報、ログイン情報、プロフィール情報等
  • 投稿・掲載依頼・レビュー・コメント・アップロードデータ等
  • アクセス日時、閲覧ページ、参照元、端末情報、ブラウザ情報、IPアドレス、Cookie識別子、広告識別子等
  • Google Analytics 等を通じて収集される利用状況データ

3. 利用目的

取得した情報は、以下の目的で利用します。

  • お問い合わせ対応、本人確認、返信のため
  • 掲載依頼、修正依頼、削除依頼への対応のため
  • 会員機能、投稿機能、保存機能等の提供および改善のため
  • サイトの品質改善、利便性向上、UI/UX改善、アクセス解析のため
  • 不正利用防止、セキュリティ確保、利用規約違反対応のため
  • 広告、提携、求人掲載等に関連する連絡や運営のため

4. Cookie 等の利用

本サイトでは、利便性向上、アクセス解析、設定保持、将来的な会員機能・広告機能の提供のため、Cookie やこれに類する技術を利用する場合があります。法令上必要な場合、解析用・広告用・機能拡張用 Cookie 等については、同意管理ツール（Cookie Banner / Consent Management Platform）を通じて事前同意を取得したうえで有効化する運用を行います。

実装の考え方
Google Analytics を EU/ドイツ向けに安全寄りで運用する場合は、同意前には GA4 を読み込まない構成にするのが実務上わかりやすいです。Cookie バナーで「同意する」を押した後にだけ GA4 スクリプトを発火させる形がおすすめです。

5. Google Analytics の利用

本サイトでは、Google が提供する Google Analytics を利用する場合があります。Google Analytics は Cookie 等を用いて、利用者による本サイトの利用状況を分析する仕組みです。収集される情報、Google による利用方法、Google のプライバシーポリシー等については Google の案内をご確認ください。

本サイトでは、必要に応じて IP 匿名化相当の配慮、内部トラフィック除外、テスト環境除外、同意取得後のみの計測開始等の措置を講じることがあります。

6. 外部サービスの利用

GitHub Pages

本サイトは GitHub Pages 上で配信される場合があります。配信基盤として GitHub 側でアクセスログその他の技術情報が処理される可能性があります。

Google Apps Script / Google Sheets

本サイトでは、データ取得や表示のために Google Apps Script、Google Sheets 等を利用する場合があります。

Google Maps / YouTube / SNS 埋め込み

地図、動画、投稿埋め込み等の表示にあたり、Google Maps、YouTube、各種SNS等の外部サービスを利用することがあります。これらの埋め込みは、表示時または操作時に第三者サービスへ通信が発生する場合があります。

7. 会員機能・投稿機能

本サイトでは、現在または将来的に会員登録、保存機能、コメント、レビュー、掲示板、掲載依頼、投稿機能等を提供する場合があります。利用者が送信・投稿した内容は、公開表示、編集、審査、削除、保管、違反対応、サービス改善等のために利用されることがあります。個人情報や公開範囲については、機能ごとの案内または設定に従うものとします。

8. 情報共有・第三者提供

本サイトは、法令上認められる場合を除き、利用者の個人情報を本人の同意なく第三者に提供しません。ただし、ホスティング、メール対応、解析、フォーム処理、埋め込み配信、技術運用等に必要な範囲で、委託先または外部サービス提供者を利用することがあります。

9. 保管期間

本サイトは、取得した情報を利用目的に必要な期間、または法令上必要な期間に限り保管します。不要となった情報については、合理的な範囲で削除または匿名化を行います。

10. 利用者の権利

適用法令により認められる範囲で、利用者は、自己に関する個人情報について、開示、訂正、削除、利用停止、同意撤回等を求めることができます。具体的な申請は Contact ページまたは下記メールアドレスまでご連絡ください。

11. 国際移転

本サイトで利用する外部サービスの一部は、EU域外の国・地域に所在する事業者により提供される場合があります。その場合、当該サービス提供者の仕組みに従って情報が域外移転される可能性があります。

12. お問い合わせ

本ポリシーに関するお問い合わせ、個人情報に関するご相談、削除・訂正・利用停止等の申請は、Contact ページからご連絡ください。
